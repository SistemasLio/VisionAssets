# VisionAssets — documentação

Site estático gerado a partir dos arquivos Markdown do [repositório](https://github.com/SistemasLio/VisionAssets). Use a **busca** (atalho `Ctrl+K`) ou a barra lateral.

## O projeto

Ferramenta para **inventário de hardware e software** em estações Windows, com **.NET**, **SQLite** local e instalação **MSI** para implantação em rede.
