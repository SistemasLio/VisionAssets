# Glossário

| Termo | Significado |
|-------|-------------|
| **Agente** | Componente que roda na máquina cliente e executa a coleta. |
| **PBI** | Product Backlog Item — item priorizável no backlog (Azure DevOps). |
| **MSI** | Pacote de instalação Windows (Microsoft Installer). |
| **WMI / CIM** | Interfaces de consulta a metadados do sistema no Windows; usadas para hardware e parte do software. |
| **SQLite** | Banco de dados relacional embutido em um arquivo local. |
| **Uninstall Registry** | Chaves do Registro onde a maioria dos instaladores registra programas para “Adicionar/remover programas”. |
| **ADR** | Architecture Decision Record — registro curto de decisão e consequências. |
| **MVP** | Minimum Viable Product — primeiro conjunto mínimo entregável. |
